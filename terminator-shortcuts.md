# Terminator Shortcuts

## Description
Terminator is terminal emulator for linux

```sh
## open new tab
ctrl + shift + t

## split window vertically
ctrl + shift + e

## split window horizontally
ctrl + shift + o

## move between split windows
alt + arrow-keys(UP, DOWN, LEFT, RIGHT)

## resize windows
ctrl + shift + arrow-keys

## full screen(zoom in) window
alt + shift + x

## close window
ctrl + d | ctrl + w
```

Terminal shortcuts

```sh
## clear screen
ctrl + l

## clear whole line on terminal
ctrl + u

## cancle the line
ctrl + c

## jump one word left
ctrl + left-arrow

## cut the word
ctrl + k

## help in finding command by keyword
ctrl + r

## edit command in vim editor
ctrl + x + e

## swaps letters from cursor
ctrl + t

## swaps words entered
esc + t 

## clear last word typed
ctrl + w

## paste last word deleted
ctrl + y

## delete terminal
ctrl + d

## exit from command
ctrl + c

## make background process
ctrl + z

## start of line
ctrl + a

## end of line
ctrl + e

## executes last entered command
$ !!

## write command with keyword specified
$ !keyword + enter

## bring last argument from recent executed command
$ !$

## take arguments from last executed command from 0 to 2
$ !:0-2

## go to last visited directory
$ cd -

## go 1 dir back
$ ..

## go 2 dir back
$ ...

## create nested dirs in one go
$ md 

## create dir and cd to it
$ take
```
